AI-Powered React UI Library
An AI-powered React component library that allows developers to use prebuilt UI components or generate custom React components using AI.
Built with the MERN stack, this platform combines reusable UI components, AI-powered generation, npm package publishing, authentication, and payment integration into one complete developer ecosystem.

✨ Features

👤 User Features:

Google Authentication using Firebase
Browse and use prebuilt React UI components
Generate custom React components using AI
Save generated components for future use
Copy clean JSX code directly into projects
Install reusable components through npm
Credit-based AI generation system
Razorpay payment integration for purchasing credits
Smooth animations using Motion


🛠️ Admin Features:
Add and manage UI components
Generate components using AI
Publish components directly to npm
Publish components through:
AI-generated code
Manual code paste
Add component props and metadata
Manage component library dynamically

🧠 AI Component Generation
Users can generate UI components using prompts like:
create a delete button
The AI generates reusable React JSX/Tailwind components instantly.
Powered by OpenRouter AI APIs.

📦 Installation
Install the component library:
npm install ganesh-ui-library

Import components:

import { Button, Card } from "ganesh-ui-library";
export default function App() {
  return (
    <div>
      <Card>
        <Button>Click Me</Button>
      </Card>
    </div>
  );
}
